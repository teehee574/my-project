Короткий гайд для Vite
1. Встановити Node.js
Переконайся, що у тебе встановлений Node.js (версія 14+).

Перевірити можна так:

bash
Копировать
Редактировать
node -v
npm -v
2. Створити новий проект з Vite
bash
Копировать
Редактировать
npm create vite@latest my-vite-app
Тебе спитають вибрати фреймворк (React, Vue, Svelte, Vanilla тощо) — вибери потрібний.

3. Перейти в папку проекту
bash
Копировать
Редактировать
cd my-vite-app
4. Встановити залежності
bash
Копировать
Редактировать
npm install
5. Запустити локальний сервер для розробки
bash
Копировать
Редактировать
npm run dev
За замовчуванням відкриється адреса: http://localhost:5173

6. Побудувати проект для продакшну
bash
Копировать
Редактировать
npm run build
7. Запустити локальний сервер для перевірки продакшн-білду
bash
Копировать
Редактировать
npm run preview
Основні команди
Команда	Опис
npm create vite@latest	Створити новий проект Vite
npm install	Встановити залежності
npm run dev	Запустити dev-сервер
npm run build	Зробити продакшн-збірку
npm run preview	Переглянути продакшн-білд локально
